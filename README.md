## About Project

Projet d'application laravel pour envoi de newsletter

## Setup Env

- `npm install && composer update`
- `cp .env.example .env`
- Configure .env file
- `php artisan key:generate`
- `php artisan migrate`
- `npm run build && php artisan serve`
